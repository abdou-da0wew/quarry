Installation Guide

This guide explains how to install Minecraft on Linux distributions including Ubuntu, Fedora, and Arch Linux.

Prerequisites

Before installing Minecraft, ensure you have:

  • Java Runtime Environment 17 or later
  • At least 4GB of RAM
  • OpenGL 4.4 compatible graphics card

Installation Steps

  1. Download the launcher from our releases page
  2. Make the file executable: chmod +x minecraft-launcher
  3. Run the launcher: ./minecraft-launcher
  4. Log in with your Mojang or Microsoft account

Distribution-Specific Notes

For Ubuntu users, you may need to install additional dependencies.

For Arch Linux users, the launcher is available in the AUR.
