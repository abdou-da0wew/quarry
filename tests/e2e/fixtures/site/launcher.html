Launcher Configuration

The Minecraft launcher can be configured to optimize performance and manage multiple installations.

Memory Settings

To allocate more memory to Minecraft, edit the launcher configuration file:

# ~/.minecraft/launcher.properties
max_memory=4G
min_memory=512M

Java Arguments

Custom Java arguments can improve garbage collection and startup time:

-Xmx4G -Xms512M -XX:+UseG1GC -XX:+ParallelRefProcEnabled

Multiple Profiles

The launcher supports multiple game profiles with different versions and mod configurations.

  • Click "Configurations" in the launcher
  • Click "New" to create a profile
  • Select the Minecraft version and any mods
  • Save and launch the profile
